# Example Hidden Skill

This is an example of what a highly-technical, long markdown skill file looks like when placed inside your hidden `~/.opencode-skill-libraries/` directory.

Because this file sits outside of `~/.config/opencode/skills/`, the AI will not load it automatically. It will only come here when the Pointer Skill directs it to.

This protects your context window!
